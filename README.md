Password Generator
A simple desktop app made with Python and Tkinter. It creates random strong passwords.
What it can do:
Choose password length
Select character types:
Uppercase letters (A-Z)
Lowercase letters (a-z)
Numbers (0-9)
Special symbols (!@#$ etc.)
Shows the generated password
Copy password with one click
